## 📝 修正步驟

### 步驟 1: 打開 Google Sheets

打開以下連結：
```
https://docs.google.com/spreadsheets/d/1VulDP7Kk_Uirag_ggRb042FI4n0BV7ntEkjsyWSI9V0/edit
```

### 步驟 2: 分享設定

1. 點擊右上角的「**分享**」按鈕（Share）
2. 在「新增使用者與群組」輸入框中，輸入服務帳號郵箱：
   ```
   phone-390@jazshun.iam.gserviceaccount.com
   ```

### 步驟 3: 設定權限

1. 點擊權限下拉選單
2. **選擇「編輯者」（Editor）** ⚠️ 重要：不是「查看者」！
3. 取消勾選「通知使用者」（因為這是機器人帳號）
4. 點擊「**分享**」或「**完成**」

### 步驟 4: 驗證權限

執行以下命令測試：

```bash
npm run test:config
```

或者手動測試編輯功能：
1. 打開後台管理系統
2. 進入「會員管理」頁面
3. 點擊某個會員的「編輯」按鈕
4. 修改方案或狀態
5. 點擊「儲存」

如果成功，表示編輯功能已啟用！

## ✅ 完成後的功能

啟用編輯權限後，你可以：

1. ✅ **編輯會員方案**（Pro/基本/免費）
2. ✅ **編輯會員狀態**（啟用/停用/已過期）
3. ✅ **透過後台直接控制 Google Sheets**
4. ✅ **所有變更會即時同步到 Google Sheets**

## ⚠️ 注意事項

- 服務帳號必須有「**編輯者**」權限才能寫入資料
- 「查看者」權限只能讀取，無法編輯
- 編輯操作會直接修改 Google Sheets 中的資料，請謹慎操作
