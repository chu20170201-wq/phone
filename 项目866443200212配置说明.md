# 项目 866443200212 配置说明

## 📋 项目信息
- **项目ID**: 866443200212

## 🔧 步骤 1: 启用 Google Sheets API

直接点击以下链接启用 API：
```
https://console.developers.google.com/apis/api/sheets.googleapis.com/overview?project=866443200212
```

点击「**启用**」按钮。

## 👤 步骤 2: 获取服务账号 JSON 文件

### 方法 A: 如果已有服务账号

1. 访问 [Google Cloud Console](https://console.cloud.google.com/)
2. 选择项目 ID: `866443200212`
3. 左侧菜单：「IAM 和管理」→「服务账号」
4. 找到或创建服务账号
5. 点击服务账号 →「密钥」标签
6. 创建新密钥（JSON 格式）
7. 下载 JSON 文件

### 方法 B: 创建新服务账号

1. 访问 [Google Cloud Console](https://console.cloud.google.com/)
2. 选择项目 ID: `866443200212`
3. 左侧菜单：「IAM 和管理」→「服务账号」
4. 点击「+ 创建服务账号」
5. 填写名称（例如：`phone-query-admin-service`）
6. 点击「创建并继续」→「完成」
7. 点击服务账号 →「密钥」标签
8. 「添加密钥」→「创建新密钥」→ 选择「JSON」
9. 下载 JSON 文件

## ⚙️ 步骤 3: 配置环境变量

下载 JSON 文件后，运行：

```bash
npm run setup
```

然后输入 JSON 文件的路径。

或者手动配置 `.env.local`：
- `GOOGLE_SHEETS_SPREADSHEET_ID`: `1VulDP7Kk_Uirag_ggRb042FI4n0BV7ntEkjsyWSI9V0`
- `GOOGLE_SERVICE_ACCOUNT_EMAIL`: 从 JSON 文件的 `client_email`
- `GOOGLE_PRIVATE_KEY`: 从 JSON 文件的 `private_key`

## 🔗 步骤 4: 分享 Google Sheets

1. 打开 Google Sheets: https://docs.google.com/spreadsheets/d/1VulDP7Kk_Uirag_ggRb042FI4n0BV7ntEkjsyWSI9V0/edit
2. 点击「分享」按钮
3. 粘贴服务账号邮箱（从 JSON 文件的 `client_email`）
4. 权限选择「查看者」
5. 点击「分享」

## ✅ 完成

配置完成后运行：
```bash
npm run test:config
```
